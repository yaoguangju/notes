1.把文件夹变成Git可管理的仓库
    git init

2.暂存区的相关操作
    git add 文件名      将文件添加到暂存区
    git add .          将全部文件添加到暂存区
    git status         查看暂存区的内容

3.上传文件到本地仓库
    git commit -m'描述内容'

4.本地仓库和github仓库的关联
    git remote add origin 新建库的地址
    关联后 git pull origin master

5.上传本地仓库内容到github仓库
    git push -u origin master

6.git的克隆操作
    git clone 地址
    
7.git中文显示8进制字符串的解决办法
    git config --global core.quotepath false
